Cara penggunaan program:
1. Run file program streamlit menggunakan VSC
2. Setelah program jalan ketik "streamlit run app.py" pada console
3. Program akan otomatis terbuka pada halaman web

Cara penggunaan Classify:
1. Tulis tweet yang ingin diklasifikasikan pada text box dalam bahasa inggris
2. Klik tombol classify
3. Hasil klasifikasi akan muncul pada bagian bawah. Jika berwarna merah artinya text / tweet termasuk kedalam depresi, jika berwarna hijau maka tidak termasuk depresi.
